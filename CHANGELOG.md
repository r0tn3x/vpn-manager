# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-09

### Added
- Initial release of VPN Manager
- Connect to VPN with simple commands (`vpn --<name>`)
- Detailed status information with uptime and traffic stats
- List available VPN configurations
- Terminate VPN connections
- Beautiful colored output with icons
- External IP detection
- Network interface monitoring
- Routing information display
- Case-insensitive config name matching
- Smart connection switching with prompts
- Comprehensive error handling
- Automatic dependency checking
- Installation script
- Complete documentation

### Features
- Support for OpenVPN configurations
- Automatic detection of .ovpn files in ~/VPN/
- Real-time traffic statistics
- Connection uptime tracking
- Multiple VPN configuration management
- Graceful and forced connection termination
- Log file management
